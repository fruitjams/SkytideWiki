Frame Hooks

These hooks are usually called every frame, You shouldnt do any blocking operations here.

Think

The seconds elapsed since the last think

Arguments

  1. number deltaSeconds
    The seconds elapsed since the last think

Example

[[[hook|hook-library.html]]].Add("Think", "DoSomething", function(deltaSeconds) print("Hello, Frames!") end)

BuildViewDrawlist

Generally called at the start of every frame when the current view wants rendering elements.

Example

[[[hook|hook-library.html]]].Add("BuildViewDrawlist", "MyRenderingCode", function() -- Render stuffs here. end)
