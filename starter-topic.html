What is Skytide?

Skytide is a game engine and platform built from the ground up with a strong emphasis on flexibility, accessibility, and modern development workflows. Its design draws clear inspiration from Garry's Mod, OD-style engines, and Roblox, which is reflected in its Lua scripting API, instance model, and hook-driven architecture.

What is Skytide Studio?

Skytide Studio is the custom editor used to create projects for the Skytide engine. It includes tools for asset management, place editing, debugging, and project management, serving as the central workspace for developing games on the Skytide platform.
